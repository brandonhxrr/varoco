# VaroCo

VaroCo is a modern, professional financial payments platform built with React Native Web. It provides a seamless experience for requesting and sending payments, tracking history, and managing user settingsâ€”all in a single-page application with a clean, Silicon Valley-inspired design.

## Features

- **Request Payments:** Easily create and manage payment requests for campaigns, personal needs, or organizations.
- **Send Payments:** Instantly transfer money to friends, family, or campaigns with a simple, secure interface.
- **Payment History:** View and track all your transactions in a clear, organized table.
- **User Settings:** Manage your profile and preferences with an intuitive settings screen.
- **Authentication:** Secure login and registration flows.

# Frontend

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository:**
   ```sh
   git clone https://github.com/brandonhxrr/varoco.git
   cd varoco/frontend
   ```

2. **Install dependencies:**
   ```sh
   npm install
   # or
   yarn install
   ```

### Running the Project

Start the development server:

```sh
npm start
# or
yarn start
```

The app will be available at [http://localhost:3000](http://localhost:3000).

### Building for Production

To build the project for production:

```sh
npm run build
# or
yarn build
```

The optimized build will be in the `build/` directory.

### Troubleshooting Build Issues

If you encounter dependency or build errors, you can try the following command to get more detailed error output and resolve issues:

```sh
npm install --legacy-peer-deps --verbose
```

This will provide more trace information and may help resolve conflicts with dependencies.
