# React Native Web Starter

Este proyecto está configurado para usar React Native y react-native-web.

## Instalación

1. Instala las dependencias:
   ```sh
   npm install
   ```
2. Inicia el servidor web:
   ```sh
   npm start
   ```

## Estructura
- `/src`: Código fuente
- `/public`: Archivos estáticos

## Scripts útiles
- `npm start`: Ejecuta la app en web
- `npm run android`: Ejecuta en Android
- `npm run ios`: Ejecuta en iOS (solo Mac)
